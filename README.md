# group-picker

Una herramienta para crear y gestionar grupos de manera sencilla.

## Descripción
Este proyecto permite organizar personas en grupos de forma rápida y eficiente. Ideal para actividades, clases, equipos de trabajo, etc.

## Instalación
```bash
npm install
```

## Uso
```bash
npm start
```
